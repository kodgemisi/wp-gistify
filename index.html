Gistify Wordpress Plugin

Wp-gistify

Download .zip Bug Report View on GitHub

What?

This is a Wordpress plugin to let you create and edit Github gist in Wordpress' bult-in editor and show them in your Wordpress posts

How?

The plugin provides a shortcode [gistify id="<gist_id>"].

In admin pages this shortcode provides a way to create and update gists
In posts and pages this shortcodes renders into embedded gists with some customization options.

FAQ

Why do you need my Github token? It is optional.

You can show any gist WITHOUT a token.

We only need Github token for creating gists on behalf of you and update your gists. We don't need a token to show any gists.

Note that you can create anonymous gist without a token but you won't be able to edit or delete them.

For further details please see: http://kodgemisi.github.io/gistify/token-help.html

What about my token security?

We don't see your token. Your token is stored in `localStorage` of your browser.

The `localStorage` is accessable from only your own machine and valid for only your own wordpress domain.
